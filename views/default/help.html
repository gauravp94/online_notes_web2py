Q1. What is the pupose of this application?
The purpose of this application is to ease up the burden of user to take notes from different online sources during a project or some class notes. This project is a very easy user interface to implement such functionality.

Q2. How to use this application
This application can be used in a variety of ways.Once can use to take online notes during surfing.
-One can add various attachments to make the project more audio-visual.
-One can schedule various tasks and reminders in his daily schedule
-Just copy the content that you want to be in the notes and just paste it in your personal and private notebook.
-After copying, one can view,edit and delete the notes which already have been created by the user.

Q3. What are the main features of the application?
The main features of this application is that :-
1. It can be accessed from anywhere in the world.
2. People can collaborate with each other by making public notes which can be viewed by your friends only.
3. A notification bar is present in the user interface which tells about the latest changes that your friend has made in his notes uptill now.
4. One can add a variety of attachments in with a particaular note.
